# Carrousel LinkedIn — Angle B "ChatGPT est un outil. Davide est un collègue."

## 1. Contexte

Campagne "Lancement de Davide", semaine 17 (mercredi 22 avril). Publication depuis le profil personnel de **Maxim Delavoet** (CEO Nando IT Solutions, voix commerciale B2B + Build in Public). Audience : dirigeants PME B2B francophones (persona Thomas — a déjà testé ChatGPT en solo, impression "gadget", peur des hallucinations). Angle B du brief — clarifier la différence entre un prompt ponctuel (outil) et un agent orchestré (collègue) sans dénigrer ChatGPT.

## 2. Framework utilisé

**Hook-Story-Offer + FAB (Features-Advantages-Benefits)**.
- **Hook-Story-Offer** structure la narration du carrousel : slide 1 = promesse qui scroll-stoppe, slides 2-3 = story (la nuance "outil vs collègue"), slides 4-7 = offer (3 bénéfices FAB + traitement objection), slide 8 = CTA.
- **FAB** structure les 3 slides de valeur (4-5-6) : on part d'une feature technique (RAG, HITL, booking client), on traduit en advantage, puis en benefit business concret pour Thomas.

Choix justifié : l'audience vit un malentendu catégoriel ("IA = prompt"). Il faut d'abord **recadrer la catégorie** (story) avant de vendre les bénéfices (offer). Le FAB oblige à ancrer chaque promesse dans une preuve technique citable.

## 9. Recommandations visuel (à transmettre au Designer)

**Format obligatoire** : carrousel LinkedIn **4:5** (1080×1350 px) ou **1:1** (1080×1080) — privilégier 4:5 pour maximiser l'emprise feed.

**Direction artistique** :
- **Palette** : bleu marine `#1E2957` (fond dominant slides 1, 3, 8) + rose nude `#F5D5CB` (fond slides 2, 4-7) pour créer un rythme visuel binaire "outil/collègue".
- **Typographie** : titres extra-bold sans-serif (Pangram Sans / Satoshi), corps sans-serif neutre (Inter).
- **Slide 1** : typographie **géante**, fond bleu marine plein, aucune image. Maximiser le scroll-stop.
- **Slides 2-3** : contraste visuel binaire — slide 2 = icône curseur/prompt stylisé (représente l'outil) ; slide 3 = icône silhouette au bureau (représente le "collègue") mais **sans visage ni robot humanoïde** (brand §5-8).
- **Slides 4-6 (FAB)** : chaque slide structurée en 3 blocs verticaux (Feature/Avantage/Bénéfice) avec code couleur progressif.
- **Slide 7** : fond contrastant pour marquer le traitement d'objection (bleu marine + accent rose sur la réponse).
- **Slide 8 (CTA)** : photo professionnelle de Maxim en bas-droite (uniquement si photo disponible, sinon logo Nando IT Solutions) + QR code Calendly optionnel.

**Interdictions designer** (brand §5 + brief §9) :
- Pas de photo-stock visible.
- Pas de robot humanoïde (contradiction totale avec le message "collègue, pas machine").
- Pas de logo client.
- Pas de chiffres "garantis" en gros — le "3x" du slide 4 doit rester dans le corps de texte attribué au témoignage, pas être sorti en titre géant (risque de lecture "promesse Nando IT Solutions").
