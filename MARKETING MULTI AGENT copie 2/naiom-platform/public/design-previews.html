3 options de design pour la plateforme

Même écran (dashboard Fireflies, vue prospect) rendu dans 3 directions distinctes. Architecture (chat, dashboards, parsers) préservée — seuls les styles changent. Clique sur l'option qui te parle, on code la direction retenue dans [naiom-platform/](naiom-platform/).

01 · visionOS Dark 02 · Liquid Glass NAIOM 03 · Bento Tahoe
OPTION 01
visionOS Dark

Fond sombre profond + aurora animée (orange, violet, rose). Cards en verre très flou, bordures 1px blanches translucides. Glow orange NAIOM. Sensation de profondeur spatiale, type Apple Vision Pro.

OPTION 02 · recommandée
Liquid Glass NAIOM

Fusion de l'identité brutaliste NAIOM (fond crème, contours noirs épais, typo Inter Black XXL, hard shadows) avec le glass iOS 26 (cards verre, backdrop-blur, transparences, parallax léger). Continuité visuelle avec les decks PDF existants.

OPTION 03
Bento Tahoe

Grille bento type macOS Tahoe avec cards colorées gradient + glass subtil. Chaque section d'info = une card avec teinte d'accent (orange, violet, vert, rose). Dashboard-first, infos denses, hover glow coloré.
